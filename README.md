# GHVAC Landing Page

Google Ads landing page draft for Giesbrecht HVAC.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Deployment

This is a Vite static site and is ready for Vercel import. Use:

- Build command: `npm run build`
- Output directory: `dist`

The form area is a placeholder until the final embedded form HTML is supplied.
